# PIXI Shapes Project

A simple interactive application built with PIXI.js, Vite, and TypeScript.

## Features
- Falling shapes (various types: Triangle, Rectangle, Pentagon, Hexagon, Circle, Ellipse, Random).
- Interactive generation on click.
- Removal on shape click.
- Controls for gravity and shape generation rate.
- Stats for number of shapes and total surface area.
- MVC and OOP architecture.

## Installation

1. Clone or download the project.
2. Ensure you have [Node.js](https://nodejs.org/) installed.
3. Install dependencies:
   ```bash
   npm install
   ```

## Development

To run the project in development mode with hot-reloading:
```bash
npm run dev
```
Open the provided URL (usually `http://localhost:5173`) in your browser.

## Build

To build the project for production:
```bash
npm run build
```
The production-ready files will be in the `dist` folder.

## Deployment

The project can be deployed to any static hosting service (e.g., GitHub Pages, Netlify, Vercel). Simply upload the contents of the `dist` folder after running the build command.
